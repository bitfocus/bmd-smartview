## Blackmagic Design SmartView / SmartScope

This module will connect to Blackmagic Design SmartView and SmartScope devices.

### Configuration
* Type in the device IP address.
* The device will communicate over port 9992.

### Available actions
* Brightness
* Contrast
* Saturation
* Identify 15 Sec
* Border
* Set Scope Type
* Audio Channels
* Set LUT
